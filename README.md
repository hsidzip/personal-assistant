🤖 AI Personal Assistant
Университетский проект: Интеллектуальный персональный помощник на базе Spring Boot, PostgreSQL и Docker.

🚀 Инструкция по запуску (для разработчика)
Если у вас уже установлен Docker, просто выполните команду в корне проекта:

Bash
docker-compose up --build
После запуска приложение будет доступно по адресу: http://localhost:8080

💻 Как запустить проект на другом компьютере (с нуля)
Чтобы запустить этот проект на другом ПК, следуйте этой пошаговой инструкции:

Шаг 1: Установка необходимых инструментов
Docker Desktop: Скачайте и установите его с официального сайта.

Важно: После установки перезагрузите компьютер и убедитесь, что Docker запущен (иконка кита в трее).

Git: Скачайте и установите Git, чтобы клонировать проект.

Шаг 2: Получение исходного кода
Откройте терминал (PowerShell или CMD) и выполните команды:

Bash
git clone https://github.com/hsidzip/personal-assistant.git
cd personal-assistant
Шаг 3: Запуск проекта
Вам не нужно устанавливать Java, Maven или PostgreSQL — всё это уже упаковано в Docker-контейнеры. Просто введите:

Bash
docker-compose up --build
Первый запуск может занять 2–5 минут, так как Docker будет скачивать необходимые образы и собирать проект.

Шаг 4: Использование
Когда в терминале появится надпись Started AssistantApplication, откройте браузер и перейдите по ссылкам:

Регистрация: http://localhost:8080/register.html

Вход: http://localhost:8080/login.html

🛠 Технологический стек
Backend: Java 17, Spring Boot 3, Spring Security (JWT)

Database: PostgreSQL

Infrastructure: Docker, Docker Compose

Frontend: HTML5, CSS3, JavaScript (Fetch API)
